# Vehicle_Rental_System-PY101-
A console-based Vehicle Rental Management System using Python OOP concepts

1. created a base vehicle class with encapsulated attributes, and then made car and bike subclasses using inheritance
2. RentalSystem class to manage vehicles, renting, returning, validations, and calculating rental costs
3. Prepared a dataset of 100 vehicles from Kaggle and stored it as JSON file
4. Added file handling to read/write vehicle data and log handling to record all rent/return actions with timestamps
5. Created backend folder with all of the initialization classes and logic code as .py files and Streamlit UI folder as well and connected them
6. Referred to Streamlit documentation for creating UI
